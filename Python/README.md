# Randomer Forest Python Package

## Requirements

1. C++ compiler (gcc)

  ```sh
  sudo apt-get install build-essential      # Ubuntu/Debian
  ```

## Installation

activate virtualenv

install from github

`pip install -e git+https://github.io/neurodata/rerf/#egg=rerf&subdirectory=Python`

build

`python setup.py build_ext --inplace`
