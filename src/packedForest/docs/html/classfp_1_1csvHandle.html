Detailed Description

fpReadCSV makes two passes over a csv file. This could be bad if the csv file is large. The first pass makes sure the csv file is structured correctly and collects basic info about the csv: number of rows and columns. The file handle is then returned to the beginning of the file and returns individual elements in order.
